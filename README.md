# Hip-tese_de_Collat_Python
hipótese matemática aplicada em Python
A hipótese de Collat se baseia nos seguintes passos: escolha um número natural não nulo. Se ele for par divida-o por 2, se for ímpar divida-o por 3 e adicione uma unidade a ele. Repita isso quantas vezes for necessário e em algum momento se chegará ao número 1. Será que você consegue descobrir um número que não se encaixa nessa "regra"?
